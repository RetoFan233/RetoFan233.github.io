# Yuanting Fan - Personal Academic Homepage

My personal academic homepage built with Jekyll and deployed on GitHub Pages, featuring a **Liquid Glass** UI design.

**Live site:** [https://retofan233.github.io/](https://retofan233.github.io/)

## Features

- Liquid Glass morphism design with frosted-glass cards and floating orbs
- Dark gradient background with dynamic lighting effects
- Scroll-triggered fade-in animations and mouse parallax interaction
- Responsive layout for desktop and mobile
- Publication management with cover images and author highlighting
- Built on Jekyll with GitHub Pages deployment

## Running Locally

1. Install [Jekyll prerequisites](https://jekyllrb.com/docs/).

2. Install dependencies:

   ```bash
   bundle install
   ```

3. Start the local server:

   ```bash
   bundle exec jekyll serve
   ```

4. Open the displayed URL in your browser.
